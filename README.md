# django-boilerplate
This is my personal Django boilerplate project, to save time when starting a new app. It has the following additions:

- Django settings pulled from environment variables.
- Django Debug Toolbar, Crispy Forms, Sentry SDK plugins installed.
- Docker and docker-compose files for development and production.
- A base template with Bootstrap v5.2.
- Bootstrap Icons installed in static files.
- jQuery installed in static files.
- 400, 403, 404 and 500 error templates.
- A custom user model that uses an email as the username field.
- Basic user authentication views: login, logout, and registration.
- Tests for the above.
